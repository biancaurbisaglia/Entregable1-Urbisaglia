Simulacion de compra en un local de hamburguesas por consola

La idea de esta simulacion es que el usuario pueda seguir una guia de pasos donde, pueda ver el menu y pueda comprar y pagar sus productos al finalizar el pedido.

Estos pasos son:

  1. El usuario pide mostrar el menu: mostrarMenu()
  2. El usuario pide un producto a traves del codigo del mismo Ej: pedirProducto(H100)
  3. El usuario pide ver el pedido realizado: verPedido()
  4. El usuario calcula el costo de su pedido: calcularCosto()
  5. El usuario finaliza su pedido: finalizarPedido()
  6. El usuario paga su pedido: pagarPedido() y en caso de pagar de mas, le damos el cambio.
